# 꼬망이들 (Little Dreamers)
카카오테크 부트캠프 2박3일 해커톤 & ICT 제주 이노베이션스퀘어 AI해커톤

## 프로젝트 소개
제주도 아동급식카드 가맹점 찾기 웹 서비스

## 주요 기능
- 카카오맵 기반 가맹점 지도 표시
- 가맹점 상세 정보 (메뉴, 가격, 리뷰)
- 이모지 기반 리뷰 시스템
- 카테고리별 필터링
- 반응형 디자인 (모바일 최적화)

## 기술 스택
- React 18.3.1
- React Router DOM 6.26.1
- Styled Components 6.1.12
- Kakao Maps API
- Vercel (배포)
